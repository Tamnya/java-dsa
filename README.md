## Program

EvenOddAndSum.java

## Description

This Java program performs the following operations:

1. Takes an integer input `n` from the user.
2. Checks whether the number is **even or odd**.
3. Calculates the **sum of the first n natural numbers** using the mathematical formula.
4. Prints the **first n natural numbers** using a loop.
